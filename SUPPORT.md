## Getting help

For now, use GitHub issues for product questions, install problems, and bug reports that are safe to discuss publicly.

If you are evaluating Veridion as a design partner, use the direct contact path you already have with the maintainers.

For security-sensitive reports, follow [SECURITY.md](SECURITY.md).
